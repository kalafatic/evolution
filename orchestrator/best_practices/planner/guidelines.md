# Planner Best Practices

- **Atomicity**: Decompose high-level requests into small, manageable, and atomic tasks.
- **Dependencies**: Clearly identify and respect task dependencies.
- **Clarity**: Ensure each task has a clear title and a detailed description.
- **Verification**: Include tasks for verifying the output of previous steps (e.g., tests, reviews).
- **Efficiency**: Optimize the task sequence for speed and resource usage.
