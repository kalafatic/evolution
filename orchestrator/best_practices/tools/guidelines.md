# Tool Usage Best Practices

- **Correct Parameters**: Ensure tool calls use the correct parameter types and formats.
- **Output Parsing**: Carefully parse and interpret the results from tool executions.
- **Retry Logic**: Be prepared to handle transient tool failures with appropriate retries.
- **Safe Execution**: Double-check destructive commands (e.g., file deletions) before execution.
- **Tool Selection**: Use the most appropriate tool for the specific task at hand.
