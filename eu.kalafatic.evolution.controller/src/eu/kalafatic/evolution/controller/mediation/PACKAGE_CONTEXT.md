# MEDIATION DOMAIN CONTEXT

## Role
Context curation and prompt synthesis for mediated LLM collaboration.

## Components
* `TargetScanner`: Discovers project artifacts.
* `SemanticExtractor`: Extracts lightweight semantic features.
* `ContextCurator`: Selects the most relevant context nodes.
* `PromptSynthesizer`: Builds optimized prompts for big LLMs.

## Evolutionary Relevance
High. This is the primary bridge between local analysis and external reasoning.
