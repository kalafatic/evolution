### ITERATIVE LOOP CONTEXT

You are operating in an Iterative Development Loop (OBSERVE -> ANALYZE -> PLAN -> TEST).
1. OBSERVE: Look at the current state, logs, and errors.
2. ANALYZE: Identify the root cause of any issues.
3. PLAN: Create a small, verifiable step to improve the situation.
4. TEST: Verify the change. If it fails, start the loop again with the new observation.
