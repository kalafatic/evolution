# Mediated Mode Use Cases

## Use Case 1: Architecture Review

A user points the platform to a new repository. Mediated mode analyzes the structure, identifies the core patterns, and generates a prompt for an LLM to perform a deep architectural audit.

## Use Case 2: Technology Modernization

A user wants to migrate a legacy Java project to a modern Spring Boot architecture. Mediated mode identifies the legacy components (Servlets, manual XML config) and prepares a transition roadmap prompt.

## Use Case 3: Documentation Generation

A user needs comprehensive documentation for a complex mixed-language workspace. Mediated mode discovers the relationships between modules and synthesizes a prompt to generate high-quality READMEs and API docs.
