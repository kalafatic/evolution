# Prompt Synthesis

Prompt Synthesis transforms the evolved understanding of a target into a high-signal prompt optimized for frontier AI models.

## Structure of a Mediated Prompt

1.  **Intent:** The original user request.
2.  **Architecture Inference:** High-level description of the system structure.
3.  **Technology Stack:** List of detected technologies and frameworks.
4.  **Context Catalog:** List of high-value files selected for the request.
5.  **Instructions:** Specific guidance on how to provide an architecturally aware response.
