# General Mediated Mode

General Mediated Mode is an evolutionary intelligence mediation layer that deeply analyzes arbitrary targets to prepare high-quality contextual cognition for large-model interaction.

## Core Philosophy

- **Mediated AI:** The platform acts as a bridge between the target and the external LLM.
- **Evolutionary Context:** Understanding of the target is evolved through multiple analysis passes.
- **Human-in-the-loop:** The user supervises and approves the discovered context and proposed prompts.

## Key Components

- **TargetScanner:** Recursively identifies files and detects technologies.
- **SemanticExtractor:** Extracts architectural patterns and semantic markers using heuristics.
- **ContextCurator:** Selects high-value context while avoiding token floods.
- **PromptSynthesizer:** Generates architecturally informed prompts based on the evolved understanding.
