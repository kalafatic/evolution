# AI SUMMARY: UI & Visualization

## What is this subsystem?
The visual representation of the evolutionary process inside Eclipse.

## Why does it exist?
To make autonomous AI reasoning transparent and controllable by human developers.

## Key Mechanisms
- **SVG Animations**: Showing data flow and network training.
- **Interactive Chat**: Directly executing kernel winners from buttons.
- **Lineage Graphs**: Visualizing the tree of competing variants.
