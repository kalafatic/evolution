# AI SUMMARY: Guidance System

## What is this subsystem?
The real-time assistant that monitors platform events to help the user.

## Why does it exist?
To lower the barrier to entry for evolutionary development and identify cognitive "dead ends" before they waste execution time.

## Interaction Map
- Listens to the **Runtime Event Bus**.
- Displays tips and warnings in the **AI Chat View**.
- Tracks **Cognitive Direction** (Velocity, Convergence).
