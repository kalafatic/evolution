# MODULE INDEX

| Module | Responsibility |
| :--- | :--- |
| **Orchestration** | Managing the 5-phase evolution loop and agent coordination. |
| **Darwin** | Spawning divergent code variants and calculating fitness scores. |
| **Mediation** | Packaging context and instructions for external LLMs. |
| **Discovery** | Recursively rebuilding the architectural model of the target project. |
| **Genome** | Persisting patterns and lessons into portable artifacts. |
| **Forge** | Designing and (eventually) training custom AI models. |
| **Guidance** | Real-time user assistance based on platform event analysis. |
| **Server** | Providing REST and database services for session state. |
| **VCS** | Automating Git operations (branches, worktrees, commits). |
| **Build** | Automating Maven and Tycho operations for verification. |
| **Cognition** | Classifying user intent and tracking cognitive direction. |
| **Supervision** | Watchdog process for kernel health and self-restarts. |
