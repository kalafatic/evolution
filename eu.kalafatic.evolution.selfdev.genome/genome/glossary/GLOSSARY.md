# GLOSSARY

- **Darwin Engine**: The multi-trajectory mutation and selection kernel.
- **Genome**: Persistent architectural memory (A: Prompts, B: Files/Metadata).
- **Iteration**: One full pass through the 5 evolutionary phases.
- **Variant**: One proposed code change trajectory within an iteration.
- **Fitness**: A metric (0.0 to 1.0) representing the quality/stability of a variant.
- **Mediation**: The process of curating context for an external AI model.
- **Orchestration**: Coordination of agents and tools to achieve a goal.
- **Supervisor**: An external watchdog process monitoring kernel health.
- **Hotspot**: An area of code identified as critical or high-risk for evolution.
- **Self-Dev**: A mode where EVO evolves its own platform source code.
- **Trajectory**: A specific architectural direction (e.g., "Service-based").
- **Blueprint**: A structured plan for a variant before it is implemented.
